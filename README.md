## tensorflow-js-hello-world

Simple hello-world example for tensorflow-js

## Prerequisites

1. Install node  
`$ brew install node`  
2. Install install node dependencies  
`$ npm install` 

## Usage

1. Build project  
`$ npm run build`  
2. Run project  
`$ npm run start` 

## Node.js & Typescript Setup

`$ npm install @tensorflow/tfjs-node`  
`$ npm install -g typescript`
